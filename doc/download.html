We have versions of NUnitAsp for NUnit 1.x and 2.x.

  • For NUnit 2.1.4, our current release is Version 1.4.1.

  • For NUnit 2.0, our current release is Version 1.4. Other than the version of NUnit, this version is identical to version 1.4.1.

  • For NUnit 1.x, our current release is Version 1.2. We recommend upgrading to NUnit 2.1.4 and using version 1.4.1 instead.

For installation instructions, see the Installation Guide. Before creating your first test, skim the checklist in the Quick Start Guide.

To be notified of new releases, subscribe to our announcements mailing list. This is a very low volume list.

For a history of changes, see the change log.

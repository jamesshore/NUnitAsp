Our current release is Version 1.5.1.

For installation instructions, see the Installation Guide. Before creating your first test, skim the Quick Start Guide.

Older Versions

The current version of NUnitAsp requires NUnit 2.2. Older versions of NUnitAsp use older versions of NUnit, but we recommend upgrading to NUnit 2.2 and using NUnitAsp 1.5.1.

  • For NUnit 2.1.4, our last release was Version 1.5.

  • For NUnit 2.0, our last release was Version 1.4.

  • For NUnit 1.x, our last release was Version 1.2.

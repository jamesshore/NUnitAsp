NUnit TestRunner Preferences

The TestRunners use a preferences file for preference settings. This file is stored in the isolated storage area for the user. The following preferences are supported:
 
filterstack Filter NUnit internal methods in failure or error stack traces. Default value= true. 
maxmessage The maximum output length. Default value= 500. Set maxmessage to -1 to disable output truncation. Disabling the output truncation will result in long test runs when an AssertEquals method fails and the ToString() method results in massive output.

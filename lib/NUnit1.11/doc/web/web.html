NUnit WebService

There is an experimental webservice NUnit.asmx in the distribution. To use it, place it in a directory, and copy the bin directory so that it is a peer of the NUnit.asmx.

Create an IIS virtual root pointed at the directory that NUnix.asmx and the bin directory are in.

Using IIS administration, give the directory script execute permission, and add a default document of NUnit.asmx.

Point your web browser to the directory holding the NUnit.asmx file (via an http://localhost/ reference, not a file:/// reference.)

You can only run tests in assemblies that are visible in the bin directory. Thus, you can run a test by typing "NUnit.Samples.SimpleTest" in the testCase parameter, then pressing Invoke.

Admire the XML output from the result. This will be compatible with the forthcoming XUnit one day.

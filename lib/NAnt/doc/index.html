NAnt : A .NET Build Tool

NAnt Home Page

NAnt is a free .NET build tool. In theory it is kind of like make without make's wrinkles. In practice it's a lot like Ant. NAnt has been tested with the release candidate .NET SDK 1.0.3705.

Getting Started

Download the latest Stable Release or a Nightly Snapshot

Note: The nigtly snapshot is not a nightly build. It only contains the full cvs repository from that night. It is not a compile from that point in time.
